## 📝 Description

**Offre Irrésistible IA** est un plugin WordPress intelligent qui aide les entrepreneurs, freelances et agences à créer ou améliorer des offres marketing percutantes.

Ce plugin propose **2 outils puissants** :
1. **Générateur d'Offres IA** : un assistant guidé pour générer des textes de vente convaincants (pour Facebook, landing pages, fiches produits) à l’aide de l’API OpenAI.
2. **Audit d’Offre IA** : collez une offre existante, obtenez une analyse complète avec ses points forts, ses faiblesses, et des conseils personnalisés.

###  Idéal pour :
*  Les infopreneurs
*  Les e-commerçants
*  Les agences marketing
*  Les freelances en quête de conversions

---

##  Fonctionnalités principales

* **Générateur d’offres guidé** (produit, prix, type d’offre, etc.)
* **Chargement dynamique** de questions marketing IA
* **Sélection et génération automatique** de *bonus à valeur perçue élevée*
* **Création de textes prêts à l’emploi** pour Facebook et le web (avec emojis et mise en forme)
* **Audit intelligent** des offres existantes : forces, faiblesses, conseils IA
* **Intégration API OpenAI GPT-4**
* **Interface claire** et responsive
* **Sécurisé** : Aucun stockage de données utilisateurs

---

##  Installation

1. **Téléchargez** le plugin ou copiez le dossier `offre-irresistible-ia` dans le répertoire `/wp-content/plugins/`.
2. **Activez-le** via le menu *Extensions* dans votre tableau de bord WordPress.
3. **Configurez** votre clé API OpenAI via la page d’options du plugin.
4. **Accédez** aux outils directement depuis l’interface admin.

---

##  Utilisation

1. Allez dans le menu **“Offre Irrésistible IA”** dans l’administration WordPress.
2. Choisissez entre le **Générateur** ou l’**Audit**.
3. Complétez les champs guidés.
4. Laissez l’IA générer ou analyser votre offre !
5. Copiez les résultats et intégrez-les dans vos campagnes ou votre site.

##  FAQ

####  Une clé API OpenAI est-elle requise ?
Oui, vous devez renseigner une clé OpenAI valide pour utiliser les outils.

####  Les textes générés sont-ils personnalisés ?
Oui. Les résultats sont entièrement personnalisés en fonction de vos réponses et de vos sélections.

####  Ce plugin stocke-t-il des données utilisateur ?
Non. Toutes les requêtes sont envoyées via AJAX et aucune donnée personnelle n’est enregistrée en base de données.

####  Est-ce compatible avec tous les thèmes ?
Oui, le plugin utilise des styles CSS sobres et flexibles compatibles avec la plupart des thèmes WordPress modernes.
